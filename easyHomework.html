Recursion Practice

1. Write a recursive function called printEven() that
takes in an int i. It should print all the even numbers
from i down to 0.

2. Write a recursive function called printAs() that takes
in an int x. It should print that number of 'A's.

3. Write a recursive function called evenness. This function
should take in an integer n and return the number of times
you can divide the number n divided by 2.
EX : evenness(8) should return 3 because 8 can be divided by 2 three times.
