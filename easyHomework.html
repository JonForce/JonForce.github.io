Inheritance Problems

Create a class called Rectangle. Rectangle should have x, y, w (for width), and h (for height).
Create a constructor that takes in four floats and sets x, y, w, and h to those floats.
Create another constructor that takes in no parameters and does nothing.

Next, create a class called Square. All squares are Rectangles, so Square should extend Rectangle.
Now, squares should have a width and height that are the same. So your next step is to create
a constructor for Square that takes in three floats as parameters, a, b, and c.
Set the x of the Square equal to a, y equal to b, and the w and h both equal to c.

Finally create a Rectangle with a width of 50 and height of 100 and then a Square with a width and height of 100.
Use the constructors to do this! Don't just create them and then use the dot operator to set the width and height.
