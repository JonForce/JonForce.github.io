Review

1.) Create an array of 100 floats.
Fill the array with random numbers.
Print all the floats in the array.
Divide every element in the array by 2.
Finally, reprint everything in the array.

2.) Create a list of Strings.

Add the String "Test" to the list 15 times.

Print the list.

Remove the fifth element from the list.

3.) Create a class called Dog.
Every dog should have a float x, y and ints r, g and b.
Create a Dog object, then in the draw function,
draw a circle at the dog's position.

4.) Create a function that takes a float and returns
that float squared.
Squared means that it should return that number times itself
