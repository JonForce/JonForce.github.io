Homework 14


Recursion Practice : Don't forget to test your code!

1. Create a recursive function called power. The function should
take in two integers, i and j. The function should return i
to the power of j.
Ex : power(2, 3) -> 8 because 2 to the power of 3 is 8

2. Create a recursive function called count that takes in an
integer i. It should print all the numbers i through 10.
Ex : count(3) should print 3 4 5 6 ... 10

3. Create a recursive function called printAll that takes in
a Stack of Strings. The function should print all the Strings in the Stack.
