Multi-Dimensional loop Problems

1.) Write a program that prints the following pattern :
1
1 2
1 2
1 2 4
1 2 4 6
1 2 4 6 8
...
1 2 4 6 8 ... 100

2.) Write a program that prints the following output :
Moose!
Moose!Moose!
Moose!Moose!Moose!
Moose!Moose!Moose!Moose!
Moose!Moose!Moose!Moose!Moose!
...
"Moose!" 100 times

Multi-Dimensional array Problems

1.) Create a 2 dimensional array of integers that is 30 by 30.

2.) Set the third element in the second array of that 2D-array to be 12345.

3.) [Hard] Loop over the 2-D array and set every element to be random.
Use a 2D Loop to do this.
Print everything in the array to be sure you did this correctly.
